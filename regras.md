# Regras do Projeto

Este arquivo é a base de conhecimento consultada pela ferramenta MCP `consultar_base_conhecimento`.

## Uso

- Edite este arquivo na raiz do projeto para definir regras, políticas ou documentação.
- O servidor MCP expõe o conteúdo via SSE/HTTP Streamable para clientes como n8n.

## Exemplo de regras

1. Manter documentação atualizada.
2. Revisar alterações antes de publicar.
3. Seguir convenções do time.
